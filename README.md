# 📔 Agenda de Contatos

Um sistema completo de gerenciamento de contatos desenvolvido com **Node.js**, **Express** e **MongoDB**. O projeto permite o cadastro, edição, visualização e exclusão de contatos, contando com um sistema de autenticação de usuários e segurança reforçada.

---

Você pode acessar o projeto online clicando no link abaixo:
👉 <https://agenda-xda4.onrender.com/>

## 🚀 Funcionalidades

* **Autenticação Completa:** Cadastro de usuários e login com sessões seguras.
* **CRUD de Contatos:** Criar, Ler, Atualizar e Deletar contatos vinculados à conta.
* **Segurança:** Utiliza **Helmet** para proteção de cabeçalhos HTTP e **CSRF Token** para proteção de formulários.
* **Mensagens de Feedback:** Sistema de Flash Messages para erros e sucessos.
* **Validação de Dados:** Garantia de e-mails válidos e campos obrigatórios.

---

## 🛠️ Tecnologias Utilizadas

* **Backend:** [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)
* **Banco de Dados:** [MongoDB](https://www.mongodb.com/) (Mongoose)
* **Template Engine:** [EJS](https://ejs.co/)
* **Bundler:** [Webpack](https://webpack.js.org/) + [Babel](https://babeljs.io/)
* **Estilização:** [Bootstrap 5](https://getbootstrap.com/)
